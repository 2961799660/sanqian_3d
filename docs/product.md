# 三千AI · 3D 完整产品介绍

## 一、产品定位

**一句话生成可商用 3D 模型**。

为以下从业者提供端到端的 AI 3D 资产生成工坊:

- **游戏美术**:NPC / 道具 / 场景元素批量出资
- **电商运营**:3D 商品上架、AR 试戴、详情页 3D 旋转
- **3D 打印 / 手办原型师**:文本 / 参考图 → 可打印模型
- **影视 / 短视频道具组**:分镜中的虚拟道具
- **印刷 / 包装设计**:产品外形快速建模
- **AR / VR 开发者**:USDZ / GLB 内容快速填充

## 二、核心能力详解

### 2.1 文本 / 图像 → 3D

**文本生 3D**:输入自然语言描述,平台自动生成对应 3D 模型。

支持的 prompt 风格(已验证):
- 物件 + 风格:`战损头盔 · 科幻`、`古董相机 · 1930 年代`、`复古录音机 · 1980 年代`
- 物件 + 材质:`雕花灯笼 · 铁艺玻璃`、`谷仓壁灯 · 黄铜工业风`
- 角色 + 姿态:`骨架人偶 · T 字形`
- 透明 / 镶嵌:`琥珀蚊子 · 琥珀包裹`
- 玩具 / 仿真:`玩具跑车 · 压铸红色`

**图像生 3D**:
- 单图直接生模
- 单图转 4 视图(前 / 后 / 左 / 右)再生模,多视角一致性更稳
- 适合实物拍摄 → 直接 3D 上架的电商场景

### 2.2 四档引擎对比

| 引擎 | 出模时间 | 拓扑质量 | 细节 | 推荐场景 |
|---|---|---|---|---|
| **P1**(旗舰) | ~30s | ★★★★★ | 最高 | 最佳几何,2026 旗舰,最终交付资产 |
| **v3.1**(高级 / H3) | ~30s | ★★★★☆ | 高 | 高级 + 全控制,贴近 prompt |
| **Turbo**(最快) | 8-12s | ★★★☆☆ | 中 | 秒级出模,质量中等,草模 / 概念验证 |
| **v1.4**(兼容) | ~10s | ★★★☆☆ | 中 | 老接口兼容,**不能进下游管线** |

**选择建议**:
- 初次试探用 Turbo 跑 prompt,锁定方向后切 P1 出最终件;
- 批量出货 v3.1 在质量和速度间最平衡;
- 历史 prompt 复刻用 v1.4 保持风格一致。

### 2.3 PBR 4K 贴图

可选项,**+10 积分**:

- **Base Color**:漫反射底色
- **Normal Map**:法线贴图,细节凹凸
- **Roughness**:粗糙度
- **Metallic**:金属度

输出分辨率 **4096 × 4096**,直接进 Unity / Unreal / Blender 标准 PBR 工作流。

### 2.4 四边面拓扑

可选项,**+5 积分**:

输出 Quad 拓扑(四边面)而非默认三角面。意义:
- 可进 Maya / Blender / ZBrush 二次雕刻 / Subdivision
- 角色类模型才能正确绑骨
- 重布线自动优化,避免长条三角面

### 2.5 自动绑骨 + 16 种预设动画

- **绑骨**:25 积分,适用人型 / 兽型角色
- **预设动画**:每个 10 积分,16 种动画分两类:
  - **双足**:站立 / 走 / 跑 / 跳 / 攻击 / 受击 / 死亡 / 待机
  - **四足**:走 / 跑 / 待机 / 跳跃 / 转身 / 进攻 / 嗅探 / 趴下

输出格式带骨骼 + 动画曲线,直接进 Unity Animator / Unreal AnimBlueprint。

### 2.6 智能处理

| 功能 | 消耗 | 用途 |
|---|---|---|
| 智能分块 | 40 积分 | 把复杂模型按结构切件,适合 3D 打印手办或可拆分配件 |
| 智能补全 | 50 积分 | 残缺扫描 / 单视角盲区自动补几何 |
| 高低模转换 | — | 同资产输出高模(展示)+ 低模(引擎实时) |

### 2.7 多格式导出

- **GLB**:Web / 引擎通用,默认推荐
- **FBX**:Maya / 3ds Max / Unreal 标准
- **USDZ**:Apple AR 生态
- **OBJ**:3D 打印 / 通用交换

**国内 GLB 加速代理**:大模型文件秒级下载,不卡 CDN。

### 2.8 失败 0 扣费

任何任务失败 —— 引擎报错、拓扑失败、贴图烘焙失败 —— 系统自动退还积分,**无需人工申诉**。

## 三、技术参数

- **任务速度**:Turbo 8-12s,P1 ~30s
- **贴图分辨率**:4K(4096×4096)
- **拓扑质量**:Quad 拓扑可重布线
- **骨骼标准**:Mixamo 兼容骨架
- **导出格式**:GLB / FBX / USDZ / OBJ
- **预览**:Web 端实时 3D 旋转预览(基于 Three.js)

## 四、典型工作流

### 工作流 1 · 游戏 NPC 资产

```
角色描述 → P1 引擎生模 → +4K PBR → +四边面拓扑 →
  自动绑骨 → 双足全套 8 动画 → 导出 GLB → 进 Unity Animator
```

单 NPC 消耗:30 + 10 + 5 + 25 + 80 = **150 积分**

### 工作流 2 · 电商 3D 商品上架

```
商品实物图 → 单图生 4 视图 → v3.1 生模 → +4K PBR →
  导出 GLB → 上架淘宝 / 京东 3D 展位
```

单 SKU 消耗:25 + 10 = **35 积分**

### 工作流 3 · 3D 打印手办

```
角色描述 → P1 生模 → 智能分块(身体 / 头 / 武器) →
  各件高低模分离 → STL 导出 → 切片打印
```

单手办消耗:30 + 40 = **70 积分**
