# 三千AI · 3D 常见问题

## 一、商用授权

**Q:产出模型可以商用吗?**
A:可以。所有产出模型 + 贴图明确授权商用,包括游戏内嵌、电商商品、印刷包装、3D 打印商品出售、影视广告等用途。

**Q:有授权证书 / 协议吗?**
A:充值即默认接受商用授权条款,如需正式授权书联系微信 aigc5256 开具。

**Q:可以基于产出再训练同类 AI 3D 模型吗?**
A:**不可以**。禁止用产出做训练集喂同类 3D 生成模型(避免左脚踩右脚)。

**Q:产出文件有水印吗?**
A:无水印,纯净文件。

## 二、质量与稳定性

**Q:能保证质量稳定吗?**
A:同一 prompt 多次出图结果会有差异,P1 引擎稳定性最高。质量不满意可"一键复刻"换种子重试。

**Q:为什么有时出来的模型不符合预期?**
A:Prompt 写法对结果影响很大。建议:
1. 加风格修饰词(`科幻 / 工业风 / 1980 年代`)
2. 加材质描述(`金属 / 玻璃 / 木质`)
3. 避免抽象描述(`一个好看的东西`)

**Q:复杂模型(多部件、镂空)效果怎么样?**
A:P1 引擎对复杂结构最稳。镂空类(雕花灯笼、骨架)建议用 P1 + 智能补全组合。

## 四、格式与导出

**Q:导出格式有哪些?**
A:GLB / FBX / USDZ / OBJ 全格式。

**Q:GLB 文件多大?会不会卡 CDN?**
A:大模型文件用国内 GLB 加速代理,秒级下载。

**Q:能直接进 Unity / Unreal 吗?**
A:可以。GLB / FBX 都进。带绑骨 + 动画的角色类资产,挂 Animator 即可用。

**Q:Mixamo 兼容吗?**
A:绑骨标准与 Mixamo 兼容,可直接用 Mixamo 动画库重定向。

**Q:3D 打印要用什么格式?**
A:OBJ 导出,切片软件读取。复杂手办建议先 +智能分块,各部件单独打印再组装。

## 五、技术能力边界

**Q:支持文本生 3D 还是图像生 3D?**
A:都支持。还可单图转 4 视图,提高多视角一致性。

**Q:能生成场景(房屋、城市)吗?**
A:目前以**单物件 / 单角色**为主,场景级生成在路线图中。

**Q:能生成超大模型吗(比如完整建筑)?**
A:单次生成有面数 / 体积上限,超大模型建议拆分多次生成后拼合。

**Q:可以重新拓扑吗?**
A:四边面拓扑选项里包含重布线,自动优化拓扑。

**Q:支持高低模分离吗?**
A:支持高低模转换,同资产输出高模(展示)+ 低模(引擎实时)双版本。
