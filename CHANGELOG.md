## [v1.x] · 2026-05

### 引擎
- **P1 引擎**(旗舰)— ~30s 出模,最佳几何,2026 旗舰
- **v3.1 引擎**(高级 / H3)— ~30s 出模,高级 + 全控制,贴近 prompt
- **Turbo 引擎**(最快)— 8-12s 出模,秒级出模,质量中等
- **v1.4 引擎**(兼容)— ~10s 出模,老接口兼容,不能进下游管线

### 交付选项
- 4K PBR 贴图(Base / Normal / Roughness / Metallic)
- 四边面 Quad 拓扑
- 自动绑骨(Mixamo 兼容)
- 16 种预设动画(双足 + 四足)
- 智能分块 / 智能补全 / 高低模转换

### 格式
- GLB / FBX / USDZ / OBJ 全格式导出
- 国内 GLB 加速代理(大文件秒级下载)

### 平台
- 失败 0 扣费保障 + 自动退款
- 新人注册即送 300 积分(价值 ¥45),14 天有效,任务失败 0 扣费
