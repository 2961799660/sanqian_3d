## Prompt 写作要点

1. **结构**:物件 + 风格 + 材质 + 年代/比例 + 关键特征
2. **避免**:抽象形容("好看的"、"酷的")
3. **加修饰词**:`磨损 / 全新 / 复古 / 现代`、`金属 / 玻璃 / 木质 / 皮革`
4. **角色姿态**:`T 字形 / A 字形 / 站立 / 蹲伏`(影响绑骨成功率)
5. **镂空类**:加 `+智能补全`,避免几何破洞
